firebase setting

To use firbase to store image, first we need to setting up in firebase

2.Selecting The Storage in left hand side, click start and you will see the following things

3.Go to the rule page, deleting the if condition behind allow read, write, and public it(ignore the warning)

4.Pod install Firebase-Core

5.Setting up the firebase in AppDelegate

Why We Should Use

The Idea of developing everythings in Component released the hard working on operation on DOM. For React, everything is state, through the state can control the view changes. But as the project keeping expends and mroe and more function are intergrated in, many states become so unmanageable that as different components trigger changes in the same state and lead to updates to the view, we may not know exactly what happened and state changes have become somewhat unpredictable and not Controlled, Redux came into being, limiting the timing and manner of stateful updates through the practice and enhancement of Flux thinking, and Redux attempted to make state changes predictable.
